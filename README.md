# news

Lightweight news/CMS application built with Yii2.

## Overview
This project includes:
- admin module
- front module
- broadcast layouts

## Tech Stack
- PHP 7.4+
- Yii2
- MySQL / MariaDB
- Redis

## Notes
- SQL dump files are not included in this repository
- Environment secrets are not committed
