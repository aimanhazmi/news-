## 文章

### 1. 查询文章列表

**接口地址**： /article/getArticleList

**请求对象**： array reqData

**响应对象**： array resData

**请求参数**： parent_id(分类ID,=0表示全部文章)

**响应参数**： total(总数);data(数据列表);page(当前页)
 
#####	请求格式：
```
[
  "parent_id"=>1,//分类ID 省略此参数或者参数为0表示查询全部
  "arti_title"=>"",//搜索关键字
  "page_no"=>2,//页码
  "page_size"=>20//每页显示数量
]

```

#####	请求格式：
```
[

]

```

[
